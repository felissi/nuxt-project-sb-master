<Meta title="Browser Compatibility" />

This project target modern browsers (browsers with ES6 modules). This project didn't transpile for any specific version of browsers. For older browsers, I recommend using polyfill.io.

In general, this project support:

- Chrome 99
- Safari 15.4
- Firefox 97
- Opera 85

If there is a need to support other version, please kindly file a issue.
